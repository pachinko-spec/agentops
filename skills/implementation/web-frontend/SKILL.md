# Web frontend実装

確認すること:

- Nuxt / Next.js の現在仕様、routing、rendering方式、build/deploy targetを公式docsで確認したか。
- static、SSG、SSR、edge runtime、client-only のどれを使うか明確か。
- UI状態、form、validation、loading、empty、error、permission denied の状態が実装されているか。
- accessibility、keyboard操作、focus、semantic HTML、responsive表示を壊していないか。
- component分割、data fetching、cache、hydration、bundle size、画像最適化の判断が妥当か。
- unit、component、E2E、ブラウザ確認、スクリーンショット確認の必要性を判断したか。
