# skills

`skills/` は、Claude Code / Codex へ採用しうる Skill 候補のカタログです。

ここに実際の `SKILL.md` 完成品は置きません。AI エージェントは `catalog.md` を読み、対象 CLI の公式 docs と実環境を確認して、必要な Skill だけを生成します。

## ファイル

- `catalog.md`: Skill 候補の名前、用途、発火条件、global/project 向き、生成時の注意。

## 生成時の確認先

- Codex: [Skills](https://developers.openai.com/codex/skills)
- Claude Code: [Skills](https://code.claude.com/docs/en/skills)
- Claude Skills standard: [Agent Skills](https://docs.claude.com/en/docs/claude-code/skills)

## 旧実体

旧 `SKILL.md` 群は `archive/reference-kit-v1/skills/` にあります。
